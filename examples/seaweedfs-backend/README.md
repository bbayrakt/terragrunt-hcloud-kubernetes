# Instructions
After setting your values in `s3.json` and deploying the Docker compose file with `docker compose up -d`, you can either

- Set up your bucket using the web interface, hosted on port 23646, or
- Set up your bucket using the [AWS CLI](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) with your SeaweedFS S3 endpoint

And adjust your Terraform backend configuration.
